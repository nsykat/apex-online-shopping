# Oracle APEX Online Shopping App

## 📌 Overview
This project is based on the Oracle LiveLabs workshop:  
[Build a Starter Online Shopping App](https://livelabs.oracle.com/ords/r/dbpm/livelabs/view-workshop?wid=848)

The application demonstrates how to build a simple e‑commerce solution in Oracle APEX, including product catalog, shopping cart, and order management. It is lightweight, demo‑ready, and includes schema setup with seed data.

## ✨ Features
- Product catalog with categories
- Shopping cart functionality
- Customer and order management
- Interactive reports and dashboards
- Schema setup with seed data
